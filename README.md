# 🔐 Password Strength Checker

A command-line tool built with Python that analyzes password strength and provides actionable feedback.

## Features
- Checks length, uppercase, lowercase, numbers, and special characters
- Detects common weak passwords
- Gives clear feedback and improvement suggestions
- Rates passwords: Weak / Moderate / Strong / Very Strong

## How to Run
```bash
python password_checker.py
```

## Built With
- Python 3
- Regex (re module)
